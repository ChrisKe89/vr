### Git

- **Install Git**
  
  ```bash
  winget install --id Git.Git -e
  ```
  
- **Verify**
  
  ```bash
  git --version
  where git
  ```
  
  **You should see something like:**

  ```bash
  C:\Program Files\Git\cmd\git.exe
  ```
